# AIJmeter
A platform for performance testing and stress testing, implemented based on Jmeter, and supports multiple testing scenarios and report analysis. Unmanned monitoring, continuous batch stress testing, generation of comprehensive stress testing results, and AI analysis report. jmeter压测平台，性能测试和压力测试的平台，基于Jmeter实现，支持多种测试场景和报表分析。形成综合压测结果。通过不同AI服务实现AI分析报告。
